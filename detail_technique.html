Méthodologie

Étape préliminaire

J'ai d'abord choisi de coder toutes mes pages html. Sur une feuille de papier, j'ai dessiné la disposition de mes pages web en m'inspirant de sites de vente bien connus comme Etsy ou Sephora. Pour chaque partie de la page, j'ai identifié à quelle balise elle correspondrait en html.

Codage des fichiers html

J'ai d'abord codé la partie html commune à toutes mes pages, le header et footer ainsi que les menus de navigation. J'ai ensuite copié ces parties sur toutes mes pages html en changeant quelques détails (par exemple, sur la page recherche, le texte s'affichant dans la barre de recherche est noir et est différent). J'ai ensuite développé chaque page html séparément. J'ai donc codé 8 pages web dont 2 pages produits différentes.

Codage des fichiers CSS

Après avoir fini le codage des fichiers html, j'ai entrepris la partie la plus longue du site, le CSS. En me basant sur ma vision initiale du site sur papier, j'ai d'abord commencé par la mise-en-forme du header et footer. J'ai principalement utilisé flexbox pour placer les éléments. Pour le footer, j'ai téléchargé sur internet du code svg représentant les réseaux sociaux. Ensuite, je me suis occupé de la mise en page de chaque page. J'ai fait 5 fichiers CSS différents en tout. Un pour le style commun à toutes les pages (header,footer) et pour les autres, j'ai associé les pages ayant une disposition proche (comme les pages d'accueil et de recherche, les pages produits ou encore les pages panier).

Pour chaque page, je m'occupais dans un premier temps de chercher les photos sur Google Images dont j'avais besoin. Ensuite, je passais à la disposition de chaque page, en utilisant flexbox ou float. Je faisais bien attention en fixant les dimensions de choisir des unités proportionnelles (em ou %). Puis, je passais aux détails de chaque page, comme les bordures, ombres, couleur de fond, taille de la police ou autre. J'ai trouvé sur internet une police permettant d'afficher des étoiles. En tapant checked, on désigne dans le code html les étoiles que l'on veut colorer. Il suffit dans le code CSS d'indiquer la couleur voulue. Cela m'a permis d'avoir un système de rating pour chacun de mes produits. Tout au long du codage CSS, j'ai dû revenir plusieurs fois sur mon code html pour ajouter des classes ou des div manquants.

Enfin, une fois satisfaite du layout de l'ensemble de mon site, j'ai travaillé à le rendre responsive sur téléphone portable. J'ai utilisé les media queries pour cela. J'ai aussi testé mon site sur des navigateurs différents comme Firefox, Chrome, Edge, et Safari pour le rendre compatible avec chacun d'entre eux. Après avoir vérifié sur W3C que mon code ne comportait pas d'erreurs, j'ai mis en ligne mon site sur Github.
